# Node.js CRUD REST API

## Fitur
- CRUD Categories & Products
- Express.js + MySQL
- Struktur MVC
- JSON Request & Response

## Cara Menjalankan
1. npm install
2. Copy .env.example menjadi .env
3. Import database.sql
4. npm start

## Endpoint
- /categories
- /products
